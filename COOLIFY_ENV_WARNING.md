# ⚠️ ВАЖНО: Настройка переменных окружения в Coolify

После деплоя этой версии **НЕОБХОДИМО** добавить переменные окружения в Coolify!

## Шаги настройки:

1. Откройте ваше приложение в Coolify
2. Перейдите в **Environment Variables** (или **Secrets**)
3. Добавьте следующие переменные:

```
ADMIN_USERNAME=ваш_логин
ADMIN_PASSWORD=ваш_пароль
```

4. **Сохраните** изменения
5. **Redeploy** приложение

## Почему это важно?

Без этих переменных приложение **НЕ ЗАПУСТИТСЯ** и будет показывать ошибку:
```
⚠️  ADMIN_USERNAME и ADMIN_PASSWORD должны быть установлены в .env файле!
```

## Уже существующие переменные в Coolify:

Убедитесь, что также установлены:
```
NODE_ENV=production
PORT=3000
DATABASE_PATH=/app/data/database.sqlite
```
